Hexo添加next主题

发表于 2020-04-11

进入 blog 目录

cd blog_folder

下载 Next 主题

git clone https://github.com/iissnan/hexo-theme-next themes/next

修改/_config.yml

  • 修改主题
  • 修改站点信息

修改/theme/next/_config.yml

  • 选择风格

    搜索 scheme，去掉所选风格前的注释

  • 设置菜单项及所对应的文件目录

    搜索 menu ，通过注释菜单项来开启或关闭菜单项

    ||后面的图标
    图标必须为 FontAwesome 网站中存在的图标的名字

  • 创建菜单项对应文件目录，以tags为例

    hexo n page tags

    编辑/source/tags/index.md，type属性的值写为tags

    • 在新创建的文章中加入 tag

  • 添加头像

    搜索 Avatar，可以用图片链接或本地图片(本地图片需要放在/source/images/下)

  • 显示浏览进度

    搜索 scrollpercent，值改为true

  • 侧边栏社交链接

    搜索 social

    ||后为图标，图标必须为 FontAwesome 网站中存在的图标的名字
